## 项目初始化：投资晴雨表网站

- 从零搭建了完整的投资晴雨表静态网站
- 技术栈: Next.js 14 (Static Export) + TailwindCSS + Python (yfinance)
- 核心功能:
  - Python脚本 (`scripts/fetch_data.py`) 抓取纳指/标普500数据、技术指标、宏观指标、板块表现、财经新闻
  - 门户首页按时间排序展示历史晴雨表
  - 详情页展示: 情绪仪表盘、RSI、MACD、布林带、均线系统、VIX、恐惧贪婪指数、板块热力图、新闻列表
  - GitHub Actions 每日美股收盘后自动抓取数据并提交
- 首次抓取数据成功: 2026-03-20, 情绪 BEARISH (-24), 纳指 22090.69, 标普 6606.49
- 部署方式: 推送到 GitHub 后 Vercel 自动导入，output 目录为 `out`
